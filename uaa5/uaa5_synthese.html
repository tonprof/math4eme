Comparatif des 3 formes du second degré

Ce tableau compare les trois écritures possibles d'une fonction polynôme du second degré. Dans tous les cas, le coefficient $a$ reste le même et détermine l'orientation de la parabole.

Caractéristiques Forme Générale Forme Canonique Forme Factorisée
Expression $$f(x) = ax^2 + bx + c$$ $$f(x) = a(x - \alpha)^2 + \beta$$ $$f(x) = a(x - x_1)(x - x_2)$$
Axe de symétrie $$x = -\frac{b}{2a}$$ $$x = \alpha$$ $$x = \frac{x_1 + x_2}{2}$$
Sommet ($S$) $$S\left(-\frac{b}{2a} ; -\frac{\Delta}{4a}\right)$$ $$S(\alpha ; \beta)$$ $$S\left(\frac{x_1+x_2}{2} ; f\left(\frac{x_1+x_2}{2}\right)\right)$$
Ordonnée à l'origine C'est la valeur $c$.
Le point est $(0 ; c)$.		 Calculer $f(0)$ :
$$a(0 - \alpha)^2 + \beta = a\alpha^2 + \beta$$		 Calculer $f(0)$ :
$$a \cdot (-x_1) \cdot (-x_2) = a \cdot x_1 \cdot x_2$$
Racines (zéros) Calculer $\Delta = b^2 - 4ac$.
  • Si $\Delta > 0$ : 2 racines :
    $$x_{1,2} = \frac{-b \pm \sqrt{\Delta}}{2a}$$
  • Si $\Delta = 0$ : 1 racine :
    $$x_0 = \frac{-b}{2a}$$
  • Si $\Delta < 0$ : Aucune.
 Résoudre :
$$a(x-\alpha)^2 + \beta = 0$$		 Lisibles directement :
$$x_1 \text{ et } x_2$$
(Ce sont les valeurs qui annulent les parenthèses).

Utilités graphiques

  • 1
    La forme générale est la plus pratique pour identifier l'ordonnée à l'origine (le point $c$) et pour calculer le discriminant $\Delta$.
  • 2
    La forme canonique est la meilleure pour tracer la courbe car elle donne immédiatement le sommet (point le plus bas ou le plus haut) et l'axe de symétrie.
  • 3
    La forme factorisée est idéale pour placer les points d'intersection avec l'axe horizontal (les racines).

Rappel sur le coefficient $a$

😊

Si $a > 0$

La parabole est "contente" (tournée vers le haut).
Le sommet est un MIN.
😞

Si $a < 0$

La parabole est "triste" (tournée vers le bas).
Le sommet est un MAX.

Les Inéquations du Second Degré

Une inéquation du second degré est une inégalité qui peut se ramener, après simplification, à l'une des formes suivantes (avec $a \neq 0$) :

$$ax^2 + bx + c > 0$$ $$ax^2 + bx + c < 0$$ $$ax^2 + bx + c \geq 0$$ $$ax^2 + bx + c \leq 0$$

La méthode générale en 3 étapes

1

Regrouper les termes

Vérifier que tous les termes sont du même côté.

Si nécessaire, rassemblez tous les termes dans le premier membre de l'inéquation pour obtenir un $0$ dans le second membre.

Ex: $x^2 > 3x - 2$
$\rightarrow x^2 - 3x + 2 > 0$
2

Signe du trinôme

Calculer $\Delta$ et faire un tableau.

Déterminez le discriminant $\Delta = b^2 - 4ac$ et les éventuelles racines ($x_1, x_2$) afin de construire le tableau de signes complet du trinôme.

3

Résoudre

Lire la solution dans le tableau.

On recherche dans le tableau les zones où l'inégalité de départ est vérifiée (les signes $+$ ou $-$). On conclut en écrivant l'ensemble des solutions $S$ (avec les intervalles).

i

Règle d'or pour le tableau de signes (Étape 2)

Pour compléter le tableau de signes du trinôme $ax^2 + bx + c$, il suffit de retenir une règle fondamentale basée sur $\Delta$ :

"Le trinôme est partout du signe de $a$, sauf entre ses éventuelles racines où il est du signe opposé."
