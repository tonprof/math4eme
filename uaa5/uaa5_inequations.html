🧭 Rappel de la méthode
  1. Préparation : S'assurer que l'inéquation est de la forme $ax^2 + bx + c > 0$ (ou $<$, $\ge$, $\le$).
  2. Racines : Calculer le réalisant $\Delta = b^2 - 4ac$ pour trouver les racines de l'équation $ax^2 + bx + c = 0$.
  3. Tableau de signes :
    • Placer les racines trouvées dans l'ordre croissant sur la première ligne.
    • Règle d'or : le signe à l'extérieur des racines est le signe de $a$. Entre les racines, c'est le signe opposé.
  4. Conclusion : Lire le tableau pour trouver les intervalles qui correspondent au signe demandé par l'inéquation initiale.
