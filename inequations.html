Méthode de résolution : Inéquations du second degré

Une inéquation du second degré est une inégalité qui peut se ramener, après simplification, à l'une des formes suivantes :

$ax^2 + bx + c > 0$ $ax^2 + bx + c < 0$ $ax^2 + bx + c \geq 0$ $ax^2 + bx + c \leq 0$

Avec $a$, $b$ et $c$ des nombres réels, et $a \neq 0$.

La méthode générale en 3 étapes

1

Regrouper les termes

Vérifier que tous les termes sont du même côté.

Si nécessaire, rassemblez tous les termes dans le premier membre de l'inéquation pour obtenir un $0$ dans le second membre.

Ex: $x^2 > 3x - 2$
$\rightarrow x^2 - 3x + 2 > 0$
2

Signe du trinôme

Calculer le discriminant ($\Delta$) et faire un tableau.

Déterminez le discriminant $\Delta = b^2 - 4ac$ et les éventuelles racines ($x_1, x_2$) afin de construire le tableau de signes complet du trinôme.

3

Résoudre

Lire la solution dans le tableau.

On recherche dans le tableau les zones où l'inégalité de départ est vérifiée (les signes $+$ ou $-$). On conclut en écrivant l'ensemble des solutions $S$ (avec les intervalles).

Règle d'or pour le tableau de signes (Étape 2)

Pour compléter le tableau de signes du trinôme $ax^2 + bx + c$, il suffit de retenir une règle fondamentale, basée sur le discriminant $\Delta$ :

"Le trinôme est partout du signe de $a$, sauf entre ses éventuelles racines où il est du signe opposé."

Exemples commentés

Exemple 1 : Avec deux racines ($\Delta > 0$)

Résoudre dans $\mathbb{R}$ : $\quad -2x^2 + 8x - 6 \geq 0$

Étape 1 : Tous les termes sont déjà à gauche. On étudie le trinôme $-2x^2 + 8x - 6$. Ici, $a = -2$, $b = 8$, $c = -6$.

Étape 2 : Signe du trinôme
Calcul de $\Delta = b^2 - 4ac = 8^2 - 4(-2)(-6) = 64 - 48 = 16$.
Comme $\Delta > 0$, il y a deux racines distinctes :
$x_1 = \frac{-b - \sqrt{\Delta}}{2a} = \frac{-8 - 4}{-4} = 3$
$x_2 = \frac{-b + \sqrt{\Delta}}{2a} = \frac{-8 + 4}{-4} = 1$

Le coefficient $a = -2$ est négatif. Le trinôme est donc du signe de $a$ ($-$) à l'extérieur des racines, et de signe opposé ($+$) entre elles.

$x$ $-\infty$ $1$ $3$ $+\infty$
Signe de $-2x^2+8x-6$ $-$ $0$ $+$ $0$ $-$

Étape 3 : Résoudre
L'inéquation initiale demande quand le trinôme est $\geq 0$ (positif ou nul). Dans le tableau, cela correspond à la zone avec le signe $+$.
Puisque l'inégalité est large ($\geq$), on inclut les racines (crochets tournés vers l'intérieur).

$S = [1 ; 3]$

Exemple 2 : Nécessité de regrouper (Pas de racine, $\Delta < 0$)

Résoudre dans $\mathbb{R}$ : $\quad 3x^2 + 5 < 2x$

Étape 1 : Regrouper les termes
On soustrait $2x$ des deux côtés pour obtenir zéro à droite :
$$3x^2 - 2x + 5 < 0$$ Ici, $a = 3$, $b = -2$, $c = 5$.

Étape 2 : Signe du trinôme
Calcul de $\Delta = (-2)^2 - 4(3)(5) = 4 - 60 = -56$.
Comme $\Delta < 0$, le trinôme n'a pas de racine réelle.
Il garde donc le signe de $a$ partout. Puisque $a = 3$ (positif), le trinôme est toujours positif.

$x$ $-\infty$ $+\infty$
Signe de $3x^2-2x+5$ $+$

Étape 3 : Résoudre
L'inéquation que l'on cherchait à résoudre (après l'étape 1) était $3x^2 - 2x + 5 < 0$ (strictement négatif).
En regardant le tableau, il n'y a aucun signe $-$. Le trinôme n'est jamais négatif.

$S = \emptyset$ (L'ensemble vide, aucune solution)

Exemple 3 : Inégalité stricte ($>$ 0) avec $\Delta > 0$

Résoudre dans $\mathbb{R}$ : $\quad x^2 - 5x + 6 > 0$

Étape 1 : L'inéquation est déjà sous la bonne forme. Ici, $a = 1$, $b = -5$, $c = 6$.

Étape 2 : Signe du trinôme
Calcul de $\Delta = (-5)^2 - 4(1)(6) = 25 - 24 = 1$.
Comme $\Delta > 0$, il y a deux racines distinctes :
$x_1 = \frac{-(-5) - \sqrt{1}}{2(1)} = \frac{5 - 1}{2} = 2$
$x_2 = \frac{-(-5) + \sqrt{1}}{2(1)} = \frac{5 + 1}{2} = 3$

Le coefficient $a = 1$ est positif. Le trinôme est donc du signe de $a$ ($+$) à l'extérieur des racines, et de signe opposé ($-$) entre elles.

$x$ $-\infty$ $2$ $3$ $+\infty$
Signe de $x^2-5x+6$ $+$ $0$ $-$ $0$ $+$

Étape 3 : Résoudre
L'inéquation demande quand le trinôme est strictement positif ($> 0$). Cela correspond aux zones avec le signe $+$.
Puisque l'inégalité est stricte, on exclut les racines (crochets tournés vers l'extérieur).

$S = ]-\infty ; 2[ \cup ]3 ; +\infty[$

Exemple 4 : Une seule racine ($\Delta = 0$)

Résoudre dans $\mathbb{R}$ : $\quad x^2 - 6x + 9 \leq 0$

Étape 1 : L'inéquation est déjà prête : $a = 1$, $b = -6$, $c = 9$.

Étape 2 : Signe du trinôme
Calcul de $\Delta = (-6)^2 - 4(1)(9) = 36 - 36 = 0$.
Comme $\Delta = 0$, il y a une seule racine (racine double) :
$x_1 = x_2 = \frac{-(-6)}{2(1)} = \frac{6}{2} = 3$

Le trinôme est du signe de $a$ (ici $a=1$, donc positif) partout sauf en sa racine où il vaut $0$. Il n'y a pas d'espace "entre" les racines pour changer de signe.

$x$ $-\infty$ $3$ $+\infty$
Signe de $x^2-6x+9$ $+$ $0$ $+$

Étape 3 : Résoudre
On cherche quand le trinôme est inférieur ou égal à $0$ ($\leq 0$).
Dans le tableau, il n'y a pas de signe $-$. Le trinôme n'est jamais strictement négatif. Cependant, il est égal à zéro en $x = 3$.
La seule solution qui vérifie l'inéquation est donc cette racine isolée.

$S = \{3\}$

Note sur les intervalles : Si l'inégalité est stricte ($<$ ou $>$), on exclut les racines de l'ensemble de solutions en tournant les crochets vers l'extérieur (ex: $]1 ; 3[$).
