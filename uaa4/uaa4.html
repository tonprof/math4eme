Tracé dynamique des fonctions

Modifiez les coefficients $a$, $b$, et $c$ puis activez les fonctions que vous souhaitez visualiser sur le repère cartésien. Cela vous permet d'observer l'impact de chaque paramètre sur l'allure des différentes familles de fonctions.

🧭 Rappel sur le rôle des paramètres
  • Le paramètre $a$ : Il influence "l'inclinaison" ou "l'ouverture" du graphique. S'il est négatif, il crée une symétrie orthogonale (un renversement) par rapport à l'axe X.
  • Le paramètre $b$ : Dans une fonction affine, c'est l'ordonnée à l'origine (le croisement avec l'axe Y). Dans le second degré, il déplace le sommet.
  • Le paramètre $c$ : Dans la parabole ($ax^2 + bx + c$), c'est l'ordonnée à l'origine (le point d'intersection avec l'axe Y).
