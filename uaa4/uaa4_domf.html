Recherche du domaine de définition ($dom f$)

Observez la fonction générée et déterminez son domaine d'existence en appliquant les règles vues en classe. Sélectionnez ensuite la bonne réponse parmi les 4 propositions.

Mémento : Rappel des conditions d'existence (C.E.)

Pour trouver le domaine de définition d'une fonction, on part du principe que $x \in \mathbb{R}$, mais on doit exclure ou limiter certaines valeurs selon ces 4 règles principales :

Règle Condition d'Existence (C.E.) Exemple type
1) Fraction
On ne peut pas diviser par zéro.		 Dénominateur $\neq 0$ $f(x) = \frac{3x+2}{2x-8}$
C.E. : $2x - 8 \neq 0 \iff x \neq 4$
$\Rightarrow \mathbf{dom f = ]-\infty ; 4[ \cup ]4 ; +\infty[}$
2) Racine d'indice pair
Une racine d'un négatif n'existe pas dans $\mathbb{R}$.		 Radicand $\geq 0$ $f(x) = \sqrt{2x+6}$
C.E. : $2x + 6 \geq 0 \iff x \geq -3$
$\Rightarrow \mathbf{dom f = [-3 ; +\infty[}$
3) Racine paire au dénominateur
Combinaison directe : ni négatif, ni nul.		 Radicand $> 0$ $f(x) = \frac{5}{\sqrt{x-1}}$
C.E. : $x - 1 > 0 \iff x > 1$
$\Rightarrow \mathbf{dom f = ]1 ; +\infty[}$
4) Cas mixte
Présence de plusieurs fractions et/ou racines.		 On combine (intersection) toutes les C.E. $f(x) = \frac{\sqrt{3-x}}{x-1}$
C.E. : $3-x \geq 0 \text{ ET } x-1 \neq 0$
$\iff x \leq 3 \text{ ET } x \neq 1$
$\Rightarrow \mathbf{dom f = ]-\infty ; 1[ \cup ]1 ; 3]}$
